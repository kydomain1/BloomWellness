# Getting Started with BloomWellness

## Quick Start

1. **Open the website**: Simply open `index.html` in your web browser
   - Chrome (recommended)
   - Firefox
   - Safari
   - Edge

2. **Navigate the site**:
   - Click on category cards to explore articles by topic
   - Use the search bar to find specific content
   - Browse all articles on the Blog page
   - Read full articles by clicking on article cards

## Website Structure

### Main Pages

- **Homepage** (`index.html`)
  - Hero section with welcome message
  - 6 category cards
  - Featured articles
  - Newsletter subscription

- **Blog Page** (`blog.html`)
  - All 6 articles
  - Category filters
  - Search functionality
  - Sorting options
  - Pagination

- **Article Pages** (`article.html?id=X`)
  - Full article content
  - Product recommendations
  - Related information
  - Author details

- **About Page** (`about.html`)
  - Company mission
  - Core values
  - Category descriptions
  - Team information

- **Contact Page** (`contact.html`)
  - Contact form
  - Contact information
  - Social media links
  - FAQ section

## Features to Try

### 1. Category Filtering
- On the blog page, click category buttons to filter articles
- Click "All Articles" to see everything

### 2. Search
- Click the search icon in the header
- Type keywords like "fashion", "skincare", or "travel"
- Press Enter or click the search button

### 3. Article Reading
- Click any article card to read the full content
- Scroll to see product recommendations
- Click "View Product" links for more details

### 4. Mobile View
- Resize your browser window to see responsive design
- On mobile, click the hamburger menu (three lines) to access navigation

### 5. Social Media
- Find social media icons in the footer
- Links open in new tabs

## Customization Tips

### Change Colors
Edit `css/style.css` and modify the CSS variables:
```css
:root {
    --primary-color: #4CAF50;  /* Change main green color */
    --secondary-color: #2E7D32; /* Change darker green */
}
```

### Add New Articles
Edit `js/articles.js` and add a new article object to the array.

### Modify Content
Edit the HTML files directly to change text, add sections, or update information.

## Troubleshooting

**Images not loading?**
- Check your internet connection
- Images are loaded from Unsplash CDN

**Search not working?**
- Make sure JavaScript is enabled in your browser
- Try refreshing the page

**Mobile menu not appearing?**
- Resize your browser to less than 768px wide
- Click the three horizontal lines in the top right
